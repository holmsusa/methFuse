# Plot method for FUSE segmentation results

Plot method for FUSE segmentation results

## Usage

``` r
# S3 method for class 'fuse_summary'
plot(x, ..., segments_to_plot = 1:50)
```

## Arguments

- x:

  A fuse_summary object

- ...:

  Additional arguments

- segments_to_plot:

  Integer vector of segment indices
